# Titanic Data Analysis and Survival Prediction 🚢

## Overview

This repository contains two projects based on the Titanic dataset:

1. **Exploratory Data Analysis (EDA)** – Understanding the dataset, cleaning data, and finding patterns.
2. **Logistic Regression Model** – Building a machine learning model to predict passenger survival.

---

## Project 1: Exploratory Data Analysis (EDA)

### Description

Performed EDA on the Titanic dataset to analyze passenger survival patterns, missing values, and relationships between features.

### Tasks Performed

* Loaded and explored the dataset
* Checked missing values
* Cleaned the data
* Analyzed statistical summary
* Created visualizations:

  * Survival count plot
  * Gender distribution
  * Age distribution histogram
  * Passenger class distribution
  * Survival by gender
  * Correlation heatmap

### Key Insights

* Female passengers had a higher survival rate
* First-class passengers were more likely to survive
* Younger passengers had slightly better survival chances

### Skills Used

* Data Cleaning
* Data Wrangling
* Data Visualization
* Exploratory Data Analysis
* Basic Statistics

## Project 2: Titanic Survival Prediction using Logistic Regression

### Description

Built a Logistic Regression model to predict passenger survival based on features like age, sex, passenger class, fare, and family details.

### Tasks Performed

* Preprocessed and cleaned the dataset
* Handled missing values
* Dropped unnecessary columns
* Encoded categorical variables
* Split data into training and testing sets
* Trained Logistic Regression model
* Evaluated model performance

### Model Evaluation

* Accuracy Score
* Confusion Matrix
* Classification Report

### Skills Used

* Machine Learning
* Logistic Regression
* Classification
* Feature Engineering
* Model Evaluation
